# Dumply — 크레딧 기반 Freemium 가격 정책

> 실측 기준일: 2026-06-21 · 환율 가정 $1 = ₩1,400 · 기본 모델 Claude Haiku 4.5 ($1 / $5 per 1M)
> 원가는 `count_tokens`(입력) + 실호출(출력)로 실측. 측정 스크립트는 비공개(키 로컬 보관).

## 1. 단위 원가 — 크레딧 1개 = "정리 1회"

"덤프 → 정리" 1회는 내부적으로 AI 2회 호출:

| 호출 | 입력 토큰 | 출력 토큰 | 비용 |
|---|---|---|---|
| `extractDump` | 1,758 | 297 | ₩4.54 |
| `moscowSuggestBatch` | 237 | 198 | ₩1.72 |
| **합계 (1 크레딧)** | | | **₩6.26** |

- 싼 모델 라우팅 시(GPT-4o-mini / Gemini Flash, 근사): **~₩0.9** (≈ 1/7)
- 비용의 절반 이상이 출력 토큰($5)이므로, 출력을 늘리는 변경은 원가에 직접 영향.

## 2. 크레딧 판매가

크레딧 1개 = AI 액션 1회(정리/분해/추정 등).

| 팩 | 크레딧 | 판매가 | ₩/크레딧 | 원가(₩6.26) | 매출총이익 | 마진 |
|---|---|---|---|---|---|---|
| Starter | 100 | ₩4,900 | 49.0 | ₩626 | ₩4,274 | 87% |
| Popular ⭐ | 500 | ₩19,900 | 39.8 | ₩3,130 | ₩16,770 | 84% |
| Pro | 1,500 | ₩49,900 | 33.3 | ₩9,390 | ₩40,510 | 81% |

- 결제수수료(PG/MoR ~5%) 차감 후 순마진 ~76~82%.
- 소액 단건 판매 금지(고정 결제수수료 희석을 위해 묶음만).
- 결제는 자체 처리(Stripe/Toss/Paddle) — Chrome은 수수료 없음. Paddle/LemonSqueezy(MoR)는 부가세·환급 자동처리 대신 ~5%.

## 3. 무료 크레딧

| 항목 | 양 | 보조 비용(₩6.26) | 목적 |
|---|---|---|---|
| 가입 웰컴(1회성) | 50~100 | ₩313~626/가입 | 첫 2~3주 체험, 첫구매 장벽 제거 |
| 매월 무료(리셋) | 월 30 | ₩188/활성유저·월 | 하루 1회꼴 — 습관 유지 + 전환 압박 |

- 라이트 유저는 무료에 머물고, 가치를 많이 보는 헤비 유저만 구매로 전환되는 모양이 목표.
- 원가가 싸서 웰컴을 100개까지 늘려 활성화율을 높여도 부담이 작음.

## 4. 손익분기 (BEP)

- 무료 유저 1명 보조 ≈ ₩188/월 (월 30크레딧).
- Popular 팩 1개 매출총이익 ₩16,770 → 무료 유저 약 89명·월 커버.
- 평균 유료 유저 이익 월 ₩4,000 가정 시, **유료 전환율 ~4.5%** 면 무료 보조 전액 자급.

## 5. 손해 안 보는 3대 안전장치

1. **BYOK(자기 API 키) 무료 유지** — 헤비 유저는 회사 비용 ₩0로 빠지므로 비용 폭발 시나리오 차단. 구조적 안전판.
2. **크레딧(회사부담) 트래픽 = 싼 모델 라우팅** — 원가 ₩6.26 → ~₩0.9, 마진 ~98%. Haiku/Sonnet은 BYOK·프리미엄용.
3. **무료 월 하드캡 + 크레딧 12개월 유효기간** — 어뷰즈·무한적립 방지.

## 6. 결론 (확정안)

- 크레딧 1개 원가 **₩6.26**, 판매 **₩33~49** (마진 81~87%)
- 무료: **웰컴 50~100 + 매월 30**, 보조 ₩188/유저·월
- 전환율 **4.5%** 면 손익분기, BYOK가 다운사이드 봉쇄

### 부록: 원가 재측정 방법

`lib/ai.js`의 프롬프트가 바뀌면 원가도 달라지므로 재측정 필요.

1. 입력 토큰: `POST /v1/messages/count_tokens` (무료) — `extractDump` 프롬프트 + 대표 덤프, `moscowSuggestBatch` 프롬프트 + 항목 JSON.
2. 출력 토큰: 동일 입력으로 실호출 1회(소액 과금) → `usage.output_tokens`.
3. 비용 = (입력×입력단가 + 출력×출력단가) / 1e6 × 환율.

모델 단가 변동 시 표의 ₩6.26과 마진을 갱신할 것.
